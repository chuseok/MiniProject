## introduction

고객이 식당의 빈자리를 실시간으로 확인하여 예약할 수 있다.

관리자가 그래프를 통해 예약 현황을 한눈에 파악할 수 있고, 인구통계학적인 정보를 제공하여 타겟층을 분석하고 마케팅 활동을 개선할 수 있다. 

웹페이지를 개발하기 위한 HTML5/CSS3를 이해하고 사용할 수 있다.


## Setup

MiniProject.sql을 다운 받아서 실행시킵니다.
